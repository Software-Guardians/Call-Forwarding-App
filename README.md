# Aura Link // PC Client

> **Advanced Call Forwarding Interface**  
> *Secure. High-Fidelity. Cyberpunk.*

Aura Link is a cutting-edge PC client designed for seamless interaction with the Aura Call Forwarding ecosystem. Built with **React** and **TypeScript**, it features a distinctive high-contrast aesthetic and robust global navigation.

## üíª getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Software-Guardians/Call-Forwarding-App.git
    cd Call-Forwarding-App/client
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Create a production build:

```bash
npm run build
```
